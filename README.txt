README FOR ANISOPTER

There is a list of dependencies in the requirements/ folder that are necessary
to run our tool but please note that this list is not complete and
additional requirements may be needed depending on your particular environment.

To run our system run server.py in src/server/ and go to address
"localhost:8090" in a browser of your choice to use the tool.

All the source code for our project is contained in src.
